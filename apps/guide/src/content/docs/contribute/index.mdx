Thank you for your interest in contributing to this site! This is a community-driven project that accepts contributions of any size and contributors of all skillsets.

:::tip
Join us on our [Discord server](https://discord.gg/the-great-outdoors-345621611770282004), which has a section dedicated to developing this website. Discord is the main place we collaborate on ideas.
:::

## Ways to Help

There are many ways to contribute to the project. If you have an idea or skillset that's not listed here, we'd love to hear from you!
