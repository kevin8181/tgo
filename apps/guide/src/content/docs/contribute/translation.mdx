## Dashboard

Our [translation dashboard](/translation) tracks and visualizes the article translation coverage. This is built with [Lunaria](https://lunaria.dev/).

:::tip
You can launch the dashboard in your local environment, reflecting your edits.

```bash
pnpm lunaria:preview
```

:::
