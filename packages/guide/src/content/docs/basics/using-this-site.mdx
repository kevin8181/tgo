The "Basics" section of this site is meant to be read beginning to end. It covers all the essential things you'll need to know and prepare before your first backpacking trip.

Next, check out the other sections for in-depth guides and intermediate/advanced topics.

### Next and Previous

At the bottom of each page, you will find links to the next and previous articles. If you're reading the guide cover-to-cover, these make it easier to navigate.

### Search

You can search the site using the search bar above, or pressing `Ctrl + K`. You can also add this as a search engine on your browser using the address `https://thegreatoutdoors.guide/warp?q=`.
